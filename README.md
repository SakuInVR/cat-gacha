# にゃんこガチャ (cat-gacha) 🐾

ボタンをクリックするだけで、世界中の可愛い猫の画像がランダムに表示されるシンプルなWebアプリケーションです。

## 🐈 作品概要
「日々の疲れを癒やしたい」という思いから作成した、癒やし系Webアプリです。
外部のAPIからリアルタイムで猫の画像を取得し、ボタンひとつで次々と新しい猫に出会える「ガチャ」のような体験を提供します。

## ✨ 特徴
- **リアルタイム画像取得**: [The Cat API](https://thecatapi.com/) を使用し、常に新しい画像を表示します。
- **直感的なUI**: シンプルなボタン操作だけで完結するデザイン。
- **レスポンシブ対応**: PCだけでなく、スマートフォンやタブレットからも快適に猫を眺めることができます。
- **スムーズな動作**: 非同期処理（`async/await`）により、ページをリロードせずに画像を切り替えます。

## 🛠 使用技術
- **HTML5**: ページの構造定義
- **CSS3**: カードデザイン、ボタンのアニメーション、レスポンシブレイアウト
- **JavaScript (ES6+)**: `fetch` APIを用いたデータ取得、DOM操作
- **The Cat API**: 猫画像の提供元

## 📝 開発のこだわり
- **ユーザー体験**: 画像が読み込まれるまでの間もレイアウトが崩れないよう、画像コンテナの最小サイズをCSSで指定しました。
- **デザイン**: 清潔感のある白を基調としたカードデザインに、猫をイメージした温かみのあるオレンジ色のアクセントカラーを採用しました。
